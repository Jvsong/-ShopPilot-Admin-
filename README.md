# ShopPilot Admin

ShopPilot Admin is a full-stack e-commerce admin system built with Spring Boot and Vue 3.

## Tech Stack

- Backend: Spring Boot, Spring Security, MyBatis-Plus, MySQL
- Frontend: Vue 3, TypeScript, Vite, Element Plus

## Features

- Product management
- Order management
- User management
- Role and permission management

## Structure

- `backend`: Spring Boot API service
- `frontend`: Vue 3 admin application
- `database`: schema and seed data
